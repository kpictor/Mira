## Trial Question

Does `roadmap-to-bottleneck-recursion` produce a better CPO / silicon photonics research package than a normal industry map by narrowing the chain to next-layer bottlenecks, stock proxy quality and falsification points?

## Current Trial Result

Positive partial.

The method improved the output by separating the visible CPO theme from three deeper bottleneck clusters:

1. External laser / InP laser manufacturing and substrates.
2. Photonic engine, COUPE and optical package integration yield.
3. Fiber and connector scale-out for million-GPU AI factories.

The package remains a watchlist and handoff, not a single-stock thesis. The next validation step is company-level work on one laser/material proxy and one packaging/connectivity proxy.

## Refresh Conditions

- NVIDIA changes CPO / Spectrum-X Photonics / Quantum-X Photonics timing.

- Coherent, Lumentum, Corning, Veeco or AXT report evidence that demand, capacity, margin or customer conversion is materially different from this map.
