# Mira Brand Assets

Avatar and project preview exports for Mira's public profile and repository
presentation.

## Files

### Avatars

- `mira-avatar-x-400.jpg` - compact X profile avatar.
- `mira-avatar-tight-512.jpg` - tight-crop avatar for social profiles.
- `mira-avatar-tight-900.jpg` - higher-resolution tight-crop avatar.
- `mira-avatar-full-512.jpg` - full composition with market backdrop.
- `mira-avatar-full-1024.jpg` - higher-resolution full composition.

### Repository Preview

- `mira-github-social-preview-title-1280x640.jpg` - recommended GitHub social preview upload.
- `mira-github-social-preview-clean-1280x640.jpg` - text-free GitHub social preview alternative.
- `mira-github-social-preview-title-640x320.jpg` - lightweight preview thumbnail.
- `mira-readme-hero-title-1600x800.jpg` - README or launch-page hero with title treatment.
- `mira-readme-hero-clean-1600x800.jpg` - text-free README or launch-page hero.
- `mira-social-square-1200x1200.jpg` - square social/share card.

All exported image files are under 1 MB.

## Creative Direction

Original owl-inspired female hero for Mira: calm, observant, nocturnal, evidence-first, with subtle AI, investment research, data analysis, and market-chart motifs.
