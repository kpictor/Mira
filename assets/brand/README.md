# Mira Brand Assets

Avatar exports for Mira's public profile.

## Files

- `mira-avatar-x-400.jpg` - compact X profile avatar.
- `mira-avatar-tight-512.jpg` - tight-crop avatar for social profiles.
- `mira-avatar-tight-900.jpg` - higher-resolution tight-crop avatar.
- `mira-avatar-full-512.jpg` - full composition with market backdrop.
- `mira-avatar-full-1024.jpg` - higher-resolution full composition.

All exported image files are under 1 MB.

## Creative Direction

Original owl-inspired female hero for Mira: calm, observant, nocturnal, evidence-first, with subtle AI, investment research, data analysis, and market-chart motifs.
