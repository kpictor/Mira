# User Preference Memory

- last_updated: 2026-05-29
- status: active
- governed_by: `MIRA.md` and `memory/MEMORY-RULES.md`

## Purpose

Record stable user preferences that help Mira produce better research outputs without weakening evidence discipline.

This file is not a diary and not a transcript summary. Do not store one-off comments, private details unrelated to research, emotional state, or speculative interpretations of user intent.

## Entry Format

Each entry should include:

- `preference`
- `scope`
- `basis`
- `last_confirmed`
- `may_override`

`may_override` must not include source quality, evidence logs, uncertainty, refresh conditions, or fact/inference/judgment separation.
