# Playbook: FOMC Language Decoder

- last_updated: 2026-05-08

## Use When

- FOMC 决议日 + Powell 记者会
- SEP 季度需要解读 dot plot 修正
- 从措辞预判未来 1-2 次会议路径

## Signal Set

- Powell 关键词鹰鸽序列：actively considering < patient < data-dependent < Skip < Hold < Pause
- dot plot 中位数 vs 上次的 bp 偏离
- dissent 数 + 方向（鹰 vs 鸽）
- 与会前 Fed Funds Futures 路径的偏离幅度
- 会前 24h 内 Nick Timiraos（WSJ）是否发文——视为 leak channel

## Pattern

- "Pause" = 未来一段时间不动，是最鹰暗示——见 2024-12 鹰派降息事件
- 鹰派降息：cut + 点阵图上修 + Powell 用 Pause/Hold → 股债双杀
- 鸽派降息：cut + Powell 用 actively considering → 风险资产涨、黄金涨
- 同方向 dissent ≥ 2 票 → 下次会议方向反转概率显著
- 流动性新闻（QT 调整 / RMP / IORB）经常盖过措辞——见 2025-12

## Interpretation

措辞是路径信号，dot plot 是量化信号；两者矛盾时以 dot plot 为准（中位数比单人措辞更代表 committee）。流动性操作出现时，单独评估其市场冲击。
