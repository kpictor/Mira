## Publication Bar

A validation run is not ready for external institutional sharing until it has:

- completed real case work, not only a plan
- source-traced evidence logs or explicit source notes
- before/after workflow review
- documented failures or workflow revisions
- no unresolved placeholders
- clear stale_after and must_refresh_if conditions
